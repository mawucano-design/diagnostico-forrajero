# Analizador Forrajero Regenerativo

Análisis satelital + ganadería regenerativa.  
Calcula biomasa, EV/ha, días de permanencia y **recomendaciones regenerativas**.

## Características
- Login por email
- Sentinel-2 (10m)
- Biomasa, NDVI, EV/ha, días
- Mapas + exportes (CSV, GeoJSON, DOCX)
- Recomendaciones regenerativas: rotación, descanso, biodiversidad

## Uso
```bash
pip install -r requirements.txt
streamlit run app.py
